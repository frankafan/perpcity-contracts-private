[perp.city](https://perp.city/) allows permissionless perpetual swaps on anything.

## Quick Start

Ensure you have `pnpm` installed

```sh
pnpm install
```

## Build

```sh
pnpm build
```

## Run Tests

```sh
pnpm test
```
